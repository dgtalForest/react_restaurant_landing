# A Restaurant Landing Page UI

## Tech Stack

**Client:** React.js

**Server:** No Backend!!

## Run Locally

Clone the project

```bash
  git clone https://github.com/dgtalforest/react_restaurant_landing
```

Go to the project directory

```bash
  cd react_restaurant_landing
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
